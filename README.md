# Cozy Chess ☕

![Status](https://img.shields.io/badge/Status-Active-success)

A beautifully crafted, cozy chess web application built with a warm, relaxing UI and smooth micro-animations. Cozy Chess elevates the classic game of strategy into a peaceful, visually stunning experience.

## ✨ Features
- **Glassmorphism Aesthetic**: A sleek, translucent interface with dynamic, floating neon backgrounds.
- **Premium UX/UI**: Beautiful gradient typography and glowing active states.
- **High-Resolution SVGs**: Crisp, scalable chess pieces.
- **Interactive Timers**: Built-in player timers with glowing pulse animations for the active turn.
- **Board Controls**: Easily flip the board orientation for local multiplayer functionality.
- **Move Highlighting**: Visual indicators for selected pieces and legal move destinations.

## 🚀 Quick Start

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Xain-Ul-Abedin/Minimalist_Chess.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Minimalist_Chess
   ```
3. Start a local HTTP server (using Python):
   ```bash
   python -m http.server 8080
   ```
4. Open your browser and navigate to `http://localhost:8080`.

## 🛠️ Technology Stack
- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with custom properties, Flexbox/Grid, and keyframe animations.
- **JavaScript (ES6)**: Game logic handling, DOM manipulation, and timer state management.
- **chess.js**: Powering the underlying chess engine, move validation, and game state tracking.
